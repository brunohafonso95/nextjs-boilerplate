# NextJS Boilerplate

This components module provides the web components that will be used on the project.

<div className="subheading">Creating a new component</div>

to create a new component you must run the following command

```bash
yarn generate
```

this command will run <a target="_blank" href="https://plopjs.com/documentation/">plop</a> and generate a new component based on plop template files,
the files that will be generated are `interfaces.ts`, `index.tsx`, `component.stories.ts` and styles.ts`, this files
contains a base of the component following the must code style in all project.
